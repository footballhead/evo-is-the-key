Evolution is the Key
================================================================================

A Ludum Dare 24 Jam entry by [44D AGAIN AGAIN](44d-again-again.itch.io).

Thanks for playing!

Instructions
--------------------------------------------------------------------------------

 1. Install the latest [Java](https://www.java.com).
 2. Double click `EvolutionIsTheKey.jar`.

Controls
--------------------------------------------------------------------------------

_NB. Can be found on start screen by typing `help`_

Where appropriate:

  * TYPE the desired command
  * Use WASD to move
  * Use MOUSE to aim
  * Hold LEFT MOUSE BUTTON to shoot
